PisoPlanner

About

If you have ever lived in a houseshare or in a space with someone else, you know that the organisation of how you share the space is not that obvious at first.
In my case as i write this i live in a houseshare with 5 other people. In order to keep the place tidy we came up with a system to clean different places each week.
However, this system became more and more difficult to keep up with since it was in form of a piece of paper on the fridge. To solve this problem i created PisoPlanner.

View project on GitHub
View application

Implementation

Before

The idea here is to create 3 pairs of two people for three tasks each week. The rotation of pairs happens each week. Forexample in the image above everybody's task moves 1 step each week. The problem with this planner is that it requires maintenance in terms of updating the agenda to keep track of the dates and who does what by paper. To be honest it's a tidious task on it's own.

After

The new planner which is in form of a web application is a reflection of the original planner but with automation. The application is equiped with backend calculations for the date and pair rotation that is triggered each week. Now all we need to do is check the site anytime and the information will be available.
