Text Summerization

About

The text summerization presented here is part of a bigger project under the name of EasyTerms, its an ongoing project that aims to facilitate the comprehension and readability of legal documents such as Terms and Condition and Privacy policies.
The solution will be presented in form of a mobile application that offers the user summarized versions of Terms and Conditions of application they are already using.

View project on GitHub

Implementation

Project Structure

The whole project consists of three main parts: The front end implemented in React Native for IOS and Android, the back end built with nodejs and supported by a Mysql database that is hosted on AWS's platforms. Last but not least the AI part which is responsible of all the NLP tasks, summarization being the main one. It is implemented in Python with Django rest framework to support the API feature. To add the bow on top, the whole project can be built with docker compose. The following image captures the overall structure.

AI

Models

The problem in question is taking long legal documents and summerize them in a way that makes them easy to read but also to understand. In order to better understand machine learning models that solve these kinds of tasks , we referred to the huggingface transformer library.
